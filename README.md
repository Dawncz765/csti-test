# 🏙️ CSTI 城格测试 —— 家乡版

一个基于 MBTI 风格设计的趣味测试网站，通过 15 道李克特量表题目，分析你家乡的**工业发展性格**，并生成 **主导城格 + 比例分布 + 复合人格分析**。

## ✨ 功能特点

- 📋 **15 道家乡特质题目**：涵盖工业、资源、旅游、科技、新能源等维度
- 🎭 **8 种城格类型**：长子、守卫、扫地僧、工厂、极客、老板、新能源儿、目的地
- 📊 **比例分布条**：展示前四名城格的得分占比，直观看到家乡的复合气质
- 💬 **复合人格分析**：根据前两名城格的差异，自动生成一段自然语言解读
- 🖼️ **动画形象占位**：支持为每个城格上传自定义动画形象（长方形/正方形均可）
- 📱 **完全响应式**：手机、平板、电脑均适配

## 🚀 快速开始

1. 下载 `index.html` 文件。
2. 创建 `images` 文件夹，放入你设计好的 8 个城格动画形象（图片格式建议 PNG，文件名必须与城格名称一致，如 `长子.png`）。
3. 直接用浏览器打开 `index.html` 即可开始测试。
4. （可选）部署到任意静态服务器（GitHub Pages、Vercel、Netlify 等）。


## ⚙️ 调整计分逻辑

权重矩阵位于 JavaScript 的 weights 数组中（约第 260 行）。每个题目对不同城格的影响系数可以手动修改，例如：

js
{ firstborn: 0.3, guardian: 1.2, factory: 0.2, ... }
数值越大表示该题目对城格影响越强。你可以根据实际教学或地域特点调整。

## 📝 测试题目

全部 15 道题目已内置，语句隐晦且不直接对应单个城格。如需增删题目，同时修改 questions 数组和 weights 数组的长度即可。
